# Xtylist Library Template

## Copy

```sh
git clone https://github.com/hlop3z/xtylist-template.git && rm -rf xtylist-template/.git
```

## Bun - Install

```sh
bun install
```

---

## Bun - Start a Component (**Element**)

**Note:** Use "`kebab-case`"

```sh
bun run start <component-name-kebab-case>
```

---

## Dev

```sh
bun run dev
```

## Build

```sh
bun run build
```

## Build `json.gz`

```sh
bun run build gz
```
